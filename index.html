What this does

Sweep doublings × feed LD × target LD, find the min-CV draft split for each

This runs a mathameticsl simulator (machineCV, machineLD, actual-vs-mechanical draft, the D1×D2×D3=K mass-balance constraint, and the min-gap grid search) across every combination of: the doubling triples you select below, the feed linear densities you list, and the target output linear densities you list. Each combination gets its own optimal (min-CV) draft split; results are ranked so you can see which doublings + feed/target pairing gives the lowest achievable final CV%.
